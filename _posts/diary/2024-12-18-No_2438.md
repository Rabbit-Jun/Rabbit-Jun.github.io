문제
--
첫째 줄에는 별 1개, 둘째 줄에는 별 2개, N번째 줄에는 별 N개를 찍는 문제  

입력
--
첫째 줄에 N(1 ≤ N ≤ 100)이 주어진다.  

내 풀이
--

```python
import sys
T= int(sys.stdin.readline())
for i in range(T):
    print('*'*(i+1))
```

다른 사람 풀이
--
```python
for i in range(int(input())):
    print((i+1)*"*")
```
